```python
"""nyan theme -- Nyan Cat with rainbow trail and twinkling stars.

Three-line theme using WALK mode. Faithful to the original meme:
pop-tart body cat, rainbow trail, starfield background.

Rendering tiers:
- emoji: Colored rainbow with emoji cat and stars
- unicode: Colored rainbow with unicode art cat and stars
- ascii: Patterned stripes (=, ~, -, .) for rainbow, text cat, asterisks for stars
"""
from __future__ import annotations

from ..models import Animation, AnimationMode, Frame
from ..registry import register_theme
```

**Design the 3-line layout:**
- Line 0 (top): starfield / sparkles
- Line 1 (middle): rainbow trail + cat body
- Line 2 (bottom): starfield / sparkles

The cat sprite is the rightmost part of the frame. The rainbow trail extends to the left. Stars are scattered in the remaining space.

**Frame art guidelines:**

For emoji tier:
- Cat body: Use a compact representation (2-3 emoji). Example: pop-tart cat could be represented as a combination of characters that evoke the meme.
- Rainbow trail: Use ANSI color codes (\033[31m for red, \033[33m for yellow, \033[32m for green, \033[36m for cyan, \033[34m for blue, \033[35m for magenta) with block characters or equal signs.
- Stars: Sparkle characters that alternate between frames for twinkle effect.

For unicode tier:
- Cat body: Box-drawing or unicode characters forming a simple cat shape
- Rainbow trail: Same ANSI colors with unicode bar characters (like \u2588 full block, \u2592 medium shade)
- Stars: Unicode stars (\u2726, \u2727, or similar) alternating for twinkle

For ASCII tier:
- Cat body: Simple text like `[=^.^=]` or similar
- Rainbow trail: Patterned stripes using =, ~, -, . characters (per CONTEXT.md locked decision). Each "stripe" uses a different character so the rainbow is visually layered even without color.
  - Example: Line 1 middle could show `====~~~~----....` repeating pattern
- Stars: Asterisks `*` and dots `.` alternating for twinkle

**CRITICAL frame rules:**
1. All frames in a tier must have exactly 3 lines
2. All lines across all frames must have the same `disp_len` (pad with spaces)
3. ANSI color codes must be closed with `\033[0m` at the end of each colored segment or line
4. The completion frame must also be 3 lines with same width
5. Test all art with `disp_len()` to ensure correct column counting

**Multiple frames for animation:**
- Create 2-4 frames that cycle:
  - Frame 0: Stars in position set A, cat pose 1
  - Frame 1: Stars in position set B, cat pose 2 (legs different)
  - (Optional Frame 2-3 for more variety)
- Stars "twinkle" by appearing in different positions across frames

**Completion frame:**
- Cat at the rightmost position
- Full rainbow trail extending the full width
- Stars sparkle around the completed scene
