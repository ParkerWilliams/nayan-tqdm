**Step 2: Write `README.md`:**

The tone should be fun, playful, and visual. Emoji welcome. The README is a marketing page first, documentation second.

Structure:
1. **Title + badges** -- PyPI version, Python versions, License, Typed badges (use shields.io URLs with nyanbar)
2. **Hero section** -- A short, punchy one-liner describing nyanbar. Include a placeholder for a GIF: `<!-- TODO: Add terminal recording GIF here -->` with a text description of what it would show.
3. **Installation** -- `pip install nyanbar`
4. **Quick Start** -- The simplest possible example: `from nyanbar import tqdm; for x in tqdm(range(100)): pass`
5. **Themes Gallery** -- All 5 themes with ASCII art previews showing what each looks like:
   - `cat_walk` -- A cat walking across the progress bar
   - `cat_bounce` -- A cat bouncing above the bar (2 lines)
   - `nyan` -- Nyan cat with rainbow trail and stars (3 lines)
   - `fish` -- Swimming fish with rising water (3 lines)
   - `rocket` -- Rocket launching with flames (4 lines)
   Show the theme name and a brief ASCII preview of what the animation looks like at ~50% progress.
6. **Examples** -- At least 5 code examples:
   - Drop-in replacement: `from nyanbar import tqdm` instead of `from tqdm import tqdm`
   - Theme selection: `for x in tqdm(range(100), theme='rocket'): ...`
   - trange shorthand: `for x in trange(100): ...`
   - Context manager: `with tqdm(total=100) as bar: bar.update(10)`
   - Manual update: `bar = tqdm(total=100); bar.update(10); bar.close()`
   - set_description: `bar.set_description("downloading")`
   - leave behavior: `tqdm(range(100), leave=False)`
   - Global theme: `set_theme('fish')`
7. **API Reference** -- Brief listing of all public exports: tqdm, trange, set_theme, list_themes, get_theme, Animation, Frame
8. **Typed** -- Note about PEP 561 py.typed support, works with mypy
9. **Requirements** -- Python 3.10+, zero dependencies
10. **License** -- MIT
