from __future__ import annotations
from .terminal import TerminalInfo, ColorTier
__all__ = ["should_use_fallback", "render_fallback_bar"]

1. **`should_use_fallback(info: TerminalInfo, disable: bool = False, position: int | None = None) -> bool`:**
   - if disable: return True
   - if not info.is_tty: return True
   - if info.color_support == ColorTier.NONE: return True
   - if info.width < 30: return True (use the info's width for the check)
   - if position is not None and position > 0: return True
   - return False

2. **`render_fallback_bar(n, total, elapsed, rate, unit, ncols, desc, postfix_str, unit_scale) -> str`:**
   - Self-contained formatting (does NOT import from formatters.py to keep modules independent for Wave 1 parallel execution). Uses simple inline helpers:
     - `_fb_format_interval(seconds)` -> MM:SS or H:MM:SS
     - `_fb_si_format(value)` -> SI-suffixed string
   - For determinate (total is not None):
     - pct = `f"{n/total*100:3.0f}%"` (handle total=0)
     - count = `f" {n_str}/{total_str}"` (SI if unit_scale)
     - time_stats = `f" [{elapsed_str}<{eta_str}, {rate_str}]"`
     - rate_str = `f"{rate:.2f}{unit}/s"` or `f"?{unit}/s"`
     - eta_str = `_fb_format_interval((total-n)/rate)` or `"?"`
     - bar_width = ncols - len(pct) - len("|") - len("|") - len(count) - len(time_stats) - len(desc_prefix) - len(postfix_suffix)
     - bar_width = max(bar_width, 10)
     - bar = `"#" * filled + " " * (bar_width - filled)`
     - Full line: `f"{desc_prefix}{pct}|{bar}|{count}{time_stats}{postfix_suffix}"`
   - For indeterminate (total is None):
     - `f"{desc_prefix}{n_str}{unit} [{elapsed_str}, {rate_str}]{postfix_suffix}"`
   - desc_prefix = `f"{desc}: "` if desc else `""`
   - postfix_suffix = `f", {postfix_str}"` if postfix_str else `""`
