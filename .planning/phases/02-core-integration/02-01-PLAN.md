```python
# nyanbar/formatters.py
from __future__ import annotations
__all__ = ["EMA", "si_format", "format_interval", "format_meter"]
```

1. **EMA class:**
   - `__init__(self, alpha: float = 0.3)` with `self._value: float | None = None`
   - `update(self, x: float)` -- if first sample, seed; else `self._value = alpha * x + (1 - alpha) * self._value`
   - `value` property returning `float | None`

2. **`si_format(value: float, precision: int = 2) -> str`:**
   - SI suffixes tuple: ("", "k", "M", "G", "T", "P", "E", "Z", "Y")
   - While `abs(value) >= 1000.0` and index < max, divide by 1000 and increment
   - Return `f"{value:.{precision}f}{suffix}"`

3. **`format_interval(seconds: float) -> str`:**
   - `m, s = divmod(int(seconds), 60)`; `h, m = divmod(m, 60)`
   - If h: `f"{h:d}:{m:02d}:{s:02d}"`; else `f"{m:02d}:{s:02d}"`

4. **`format_meter(n, total, elapsed, rate, unit, ncols, desc, postfix_str, unit_scale, unit_divisor) -> str`:**
   - If `total` is not None (determinate):
     - `pct = n / total * 100` (guard division by zero)
     - Build count string: `f" {n}/{total}"` (or SI-formatted if unit_scale)
     - Build time string: `f" [{format_interval(elapsed)}<{eta_str}, {rate_str}]"`
     - Build bar: `f"{pct:3.0f}%|{bar_fill}| "` where bar_fill is `#` * filled + ` ` * remaining, sized to fit ncols after accounting for prefix/suffix widths
     - Rate string: `f"{rate:.2f}{unit}/s"` or `f"?{unit}/s"` if rate is None
     - ETA: `format_interval((total - n) / rate)` or `"?"` if rate is None
   - If `total` is None (indeterminate):
     - `f"{n_str}{unit} [{format_interval(elapsed)}, {rate_str}]"`
   - Prepend `desc + ": "` if desc is not None
   - Append `", " + postfix_str` if postfix_str is not None
   - If unit_scale: format n, total, and rate with `si_format()`
   - ncols parameter controls total line width (default 80); bar portion fills remaining space
