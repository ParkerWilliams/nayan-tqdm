**Init body:**
- `self.iterable = iterable`
- `self.desc = desc`
- `self.total = total` -- if None and iterable has `__len__`, infer: `self.total = len(iterable)`
- `self.leave = leave`
- `self.file = file or sys.stderr`
- `self.ncols = ncols`
- `self.mininterval = mininterval`
- `self.maxinterval = maxinterval`
- `self._miniters = miniters if miniters is not None else 1`
- `self.disable = disable`
- `self.unit = unit`
- `self.unit_scale = unit_scale`
- `self.dynamic_ncols = dynamic_ncols`
- `self.smoothing = smoothing`
- `self.position = position`
- `self.unit_divisor = unit_divisor`
- `self.n = initial`
- `self._desc = desc`
- `self._postfix_str: str | None = None`
- If postfix dict provided: call `self.set_postfix(postfix)`
- `self._closed = False`
- `self._lock = threading.Lock()`
- `self._start_t = time.monotonic()`
- `self._last_print_t = self._start_t`
- `self._last_print_n = initial`
- `self._ema_dn = EMA(alpha=smoothing)`
- `self._ema_dt = EMA(alpha=smoothing)`
- `self._prev_height = 0`
- Terminal detection: `self._terminal = detect_terminal(self.file)`
- Fallback decision: `self._use_fallback = should_use_fallback(self._terminal, disable=self.disable, position=self.position)`
- `self._anim_thread: threading.Thread | None = None`
- `self._running = False`
- If not disable: render initial state (display the 0% bar)

**`__iter__` method:**
- If iterable is None: raise TypeError
- If disable: yield from iterable, return
- Inline hot-path local variables for performance (per research):
  ```python
  mininterval = self.mininterval
  miniters = self._miniters
  last_print_n = self._last_print_n
  last_print_t = self._last_print_t
  n = self.n
  ```
- For each item: yield, increment n, check iteration gate (n - last_print_n >= miniters), then time gate (dt >= mininterval), then refresh display
- After loop: `self.n = n; self.close()`

**`update(n=1)` method:**
- Acquire lock, increment `self.n += n`
- Check two-gate throttle (miniters + mininterval)
- If gates pass: refresh display, update last_print_n/t, auto-tune miniters
- Release lock

**`close()` method:**
- Guard: `if self._closed: return`
- Set `self._closed = True`
- Stop animation thread if running (set `self._running = False`)
- If disable: return
- If `self.leave`:
  - Render final bar state (100% or current state)
  - Write newline
- Else:
  - Erase all rendered lines using `erase_lines(self.file, self._prev_height)` from renderer.py
- Flush file

**`__enter__` / `__exit__`:**
- `__enter__`: return self
- `__exit__`: call `self.close()` unconditionally (regardless of exc_type); return False (don't suppress exceptions)

**`set_description(desc, refresh=True)`:**
- `self._desc = desc`
- If refresh and not closed and not disable: trigger display refresh

**`set_postfix(ordered_dict=None, refresh=True, **kwargs)`:**
- Merge ordered_dict and kwargs
- Build `self._postfix_str` as `", ".join(f"{k}={v}" for k, v in items.items())`
- If refresh and not closed and not disable: trigger display refresh

**`set_postfix_str(s, refresh=True)`:**
- `self._postfix_str = s`
- If refresh: trigger display refresh

**`_refresh()` internal method:**
- Compute elapsed = `time.monotonic() - self._start_t`
- Compute rate from EMA: `rate = self._ema_dn.value / self._ema_dt.value` if both exist, else None
- If `self._use_fallback`:
  - Call `render_fallback_bar(...)` to get line string
  - If TTY: write `\r` + line (carriage return overwrite)
  - If not TTY: write line + `\n` (newline for piped output)
- Else:
  - For now (pre-Plan 04), also use fallback rendering as placeholder. Plan 04 will add animation rendering.
- Flush file

**`_effective_width()` helper:**
- If `self.ncols` is set: return `self.ncols`
- If `self.dynamic_ncols`: re-query terminal width
- Else: return `self._terminal.width`
- Reserve 1 column to prevent line wrap: `return max(width - 1, 10)`

**`_auto_tune_miniters(dn, dt)` helper:**
- EMA-based: `self._miniters = max(1, int(self.smoothing * dn * (self.mininterval / dt) + (1 - self.smoothing) * self._miniters))`

**`write()` classmethod:**
```python
@classmethod
def write(cls, s: str, file: IO[str] | None = None, end: str = "\n") -> None:
    fp = file if file is not None else sys.stdout
    fp.write("\r\033[2K")  # CR + erase line
    fp.write(s + end)
    fp.flush()
```

```python
"""nyanbar - Animated progress bars for Python.

"""
from __future__ import annotations

from .core import NyanBar
from .core import NyanBar as tqdm
from .models import Animation, AnimationMode, Frame

__all__ = [
    "NyanBar",
    "tqdm",
    "trange",
    "Animation",
    "AnimationMode",
    "Frame",
]


def trange(n: int, *args, **kwargs) -> NyanBar:
    """Shorthand for ``NyanBar(range(n), ...)``."""
    return NyanBar(range(n), *args, **kwargs)
```
