<feature>
  <name>Terminal Capability Detection</name>
  <files>nyanbar/terminal.py, tests/test_terminal.py</files>
  <behavior>
    ColorTier enum:
    - TRUE_COLOR = 3 (24-bit)
    - COLOR_256 = 2 (256 colors)
    - COLOR_16 = 1 (basic ANSI 16 colors)
    - NONE = 0 (no color support)

    TerminalInfo frozen dataclass:
    - is_tty: bool
    - color_support: ColorTier
    - width: int
    - is_notebook: bool

    detect_terminal(stream=None) -> TerminalInfo:
    - Default stream: sys.stderr (tqdm convention)
    - is_tty: stream.isatty() if hasattr, else False
    - width: shutil.get_terminal_size().columns if is_tty, else 80
    - color detection (only if is_tty):
      - NO_COLOR env var set -> NONE
      - TERM=dumb -> NONE
      - COLORTERM in (truecolor, 24bit) -> TRUE_COLOR
      - TERM contains "256color" -> COLOR_256
      - TERM is non-empty -> COLOR_16
      - else -> NONE
    - notebook detection:
      - Try IPython.get_ipython().__class__.__name__ == 'ZMQInteractiveShell'
      - ImportError or NameError -> is_notebook=False
    - Non-TTY always returns color_support=NONE regardless of env vars

    Test cases (use mocking for os.isatty, os.environ, shutil.get_terminal_size, IPython):
    - TTY with xterm-256color -> is_tty=True, COLOR_256
    - TTY with COLORTERM=truecolor -> TRUE_COLOR
    - TTY with TERM=dumb -> NONE
    - Non-TTY (piped) -> is_tty=False, NONE, width=80
    - NO_COLOR set -> NONE regardless of TERM
    - Notebook (ZMQInteractiveShell) -> is_notebook=True
    - No IPython -> is_notebook=False
    - Terminal width 40 -> width=40
    - NYANBAR_DISABLE=1 is NOT handled here (that's Phase 2 NyanBar logic)
  </behavior>
  <implementation>
    Use @dataclass(frozen=True, slots=True) for TerminalInfo.
    Use enum.IntEnum for ColorTier (allows comparison: ColorTier.TRUE_COLOR > ColorTier.COLOR_16).
    detect_terminal() is a pure function — takes optional stream, reads env, returns TerminalInfo.
    Tests use unittest.mock.patch for os.environ, sys.stdout.isatty, shutil.get_terminal_size.
    Notebook detection wrapped in try/except to avoid IPython import errors.
  </implementation>
</feature>
