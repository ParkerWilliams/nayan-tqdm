<feature>
  <name>Display Width and ANSI Utilities</name>
  <files>nyanbar/utils.py, tests/test_utils.py</files>
  <behavior>
    disp_len(text: str) -> int:
    - ASCII text: each char = 1 column. "hello" -> 5
    - East Asian Wide (W) and Fullwidth (F) chars: each = 2 columns. "\u4e16" (CJK) -> 2
    - Emoji: each = 2 columns. "\U0001f600" -> 2
    - Combining characters: 0 columns. "e\u0301" (e + accent) -> 1
    - ANSI escape sequences: 0 columns. "\033[31mhi\033[0m" -> 2
    - Mixed: "\033[31m\u4e16界\033[0m" -> 4
    - Empty string: 0
    - NFC normalization applied before measurement

    strip_ansi(text: str) -> str:
    - Removes all SGR sequences: "\033[31mhello\033[0m" -> "hello"
    - Removes cursor movement sequences: "\033[2K\033[1A" -> ""
    - Preserves non-ANSI text intact
    - Handles nested/adjacent sequences
  </behavior>
  <implementation>
    Use unicodedata.east_asian_width() for character width. W and F = 2, all others = 1.
    Use unicodedata.combining() to detect combining chars (width 0).
    Use unicodedata.normalize('NFC', text) before measurement.
    Use re.compile(r'\033\[[0-9;]*[a-zA-Z]') for ANSI stripping (covers SGR, cursor, erase).
    strip_ansi called inside disp_len before character iteration.
    Also create nyanbar/__init__.py (empty for now) and nyanbar/py.typed (empty marker).
    Also create tests/__init__.py (empty).
  </implementation>
</feature>
