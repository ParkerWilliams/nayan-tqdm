<feature>
  <name>ANSI Frame Renderer</name>
  <files>nyanbar/renderer.py, tests/test_renderer.py</files>
  <behavior>
    Constants:
    - CURSOR_UP = "\033[{n}A"  (move cursor up n lines)
    - ERASE_LINE = "\033[2K"   (erase entire current line)
    - RESET = "\033[0m"        (reset all attributes)
    - CR = "\r"                (carriage return)

    erase_lines(stream: IO[str], count: int) -> None:
    - If count <= 0: do nothing (no-op)
    - Move cursor up `count` lines
    - For each line: write ERASE_LINE + newline (except last)
    - Move cursor back up to start position
    - Write CR to return to column 0

    render_frame(lines: Sequence[str], prev_height: int, stream: IO[str]) -> int:
    - If prev_height > 0: call erase_lines(stream, prev_height)
    - Write each line followed by newline
    - Flush stream
    - Return len(lines) as new height

    ensure_reset(line: str) -> str:
    - If line contains any \033[...m sequence but does not end with \033[0m: append \033[0m
    - If line has no ANSI sequences: return unchanged
    - If line already ends with \033[0m: return unchanged

    has_unclosed_ansi(text: str) -> bool:
    - Parse all \033[...m sequences in text
    - Track open state: any non-reset SGR opens, \033[0m closes
    - Return True if text ends in open state
    - Used for verification/testing, not in hot path

    Test cases (capture stream output with io.StringIO):
    - First render (prev_height=0): no cursor-up emitted, lines written, returns height
    - Second render (prev_height=3): cursor-up 3, erase 3 lines, new lines written
    - Height transition 3->2: erase 3, write 2, no ghost line
    - Height transition 1->3: erase 1, write 3
    - ensure_reset adds reset to colored line
    - ensure_reset no-ops on plain line
    - ensure_reset no-ops on already-reset line
    - has_unclosed_ansi detects open red: "\033[31mhello"
    - has_unclosed_ansi returns False for "\033[31mhello\033[0m"
    - erase_lines with count=0 writes nothing
    - render_frame flushes the stream
  </behavior>
  <implementation>
    All functions are pure/stateless — they take a stream and data, produce output.
    No classes needed — renderer is a collection of functions.
    ANSI constants defined as module-level strings.
    ensure_reset applied per-line inside render_frame before writing.
    Use re module for ANSI sequence detection in has_unclosed_ansi.
    Tests use io.StringIO as mock stream to capture exact escape sequence output.
    Import strip_ansi from .utils if needed for width-related operations.
  </implementation>
</feature>
